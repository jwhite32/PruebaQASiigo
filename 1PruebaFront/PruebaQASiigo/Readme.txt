EJERCICIO FrontEnd.

A continuacion se describe, como ejecutar la solución del ejercicio de Front. Se utilizo el patro POM.

1) Importar proyecto en cualquier IDE.(Si es posible en Eclipse)
2) Después de importar el proyecto, visualizarlo, verificar que las dependencias del archivo POM se encuentren.
3) Seleccionar carpeta "src/test/java"
4) Visualizar folder test, seleccionarlo. Se visualizan dos archivos: TestRunner.java 
5) Ejecutar archivos TestRunner.java con Junit, visualizar la automatización del login del sitio: https://qastaging.siigo.com/#/login

NOTA: Se dificulto encontrar los elementos del botón Crear y del formulario de Crear Cliente, por el shadow POM.
